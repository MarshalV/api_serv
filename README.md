# D&E API (Department & Employee API)

Это сервис для управления структурой организации (отделами и сотрудниками), построенный на базе FastAPI.

## Основные возможности

- **Управление отделами**: создание, обновление, получение дерева отделов с настраиваемой глубиной вложенности.
- **Управление сотрудниками**: добавление сотрудников в конкретные отделы.
- **Гибкое удаление**: возможность каскадного удаления отделов или переназначения сотрудников в другой отдел при удалении.
- **Документация**: автоматическая генерация Swagger/ReDoc.

## Стек технологий

- **Backend**: Python 3.12, FastAPI
- **Database**: PostgreSQL 15
- **ORM**: SQLAlchemy
- **Migrations**: Alembic
- **Proxy**: Nginx
- **Containerization**: Docker, Docker Compose

## Быстрый запуск

Для запуска всего проекта (API + БД + Nginx) выполните команду:

```bash
docker-compose up -d --build
```

После запуска сервисы будут доступны по следующим адресам:
- **API (через Nginx)**: [http://localhost](http://localhost)
- **Swagger UI**: [http://localhost/docs](http://localhost/docs)
- **Direct API access**: [http://localhost:8000](http://localhost:8000)

## Настройка окружения

Проект использует файл `.env` для хранения конфигурации. Убедитесь, что он настроен корректно (пример параметров есть в `docker-compose.yaml`).

Тестовый `.env` можно найти в папке `test_.env`
